# Sample Character: Mara Vale

This is the canonical static sample character for Hunin's first guest flow. Mara Vale is original, setting-neutral, and safe to include in a public repository.

Treat this as display content for the first static guest experience, not as the final canonical character-data model.

## Character identity

- Name: Mara Vale
- Ancestry: Human
- Class: Ranger
- Subclass: Hunter
- Level: 3
- Background: Outlander
- Alignment: Neutral Good
- Character concept: A calm wilderness scout and practical guide. Mara is observant, prepared, and steady under pressure. She is not based on or connected to any existing D&D setting or published character.

## Why Mara is the sample character

Mara demonstrates the parts of Hunin's first guest flow that matter most:

- a clear primary attack;
- useful spells;
- concentration;
- an ability with a simple "How does this work again?" explanation;
- readable combat stats;
- expandable Actions, Features, and Spells sections;
- some empty sections that can remain hidden.

### Features

Features are shown as an expandable section in the first sample Character Reference.

#### Archery

- Type: Fighting Style
- Short effect: Gain +2 to attack rolls made with ranged weapons.
- Quick-reference summary: This bonus is already included in Mara's longbow attack bonus.

#### Colossus Slayer

- Type: Hunter feature
- Short effect: Once per turn, when Mara hits a creature that is below its hit point maximum, the attack deals an extra 1d8 damage.
- Quick-reference summary: After you hit an already injured enemy, add 1d8 damage. Only once each turn.
- Timing: Once per turn
- Resource: No limited resource

### Spells

Spells are shown as an expandable section in the first sample Character Reference.

#### Hunter's Mark

- Level: 1
- Action type: Bonus Action
- Casting time: Bonus Action
- Duration: Up to 1 hour
- Concentration: Yes
- Short effect: Mark one creature. Mara deals an extra 1d6 damage whenever she hits it with a weapon attack.
- Quick-reference summary: Mark a target, then add 1d6 damage to each weapon hit against it. Moving the mark to a new target later also uses a Bonus Action.
- Spell slot: 1st-level spell slot

#### Fog Cloud

- Level: 1
- Action type: Action
- Casting time: Action
- Duration: Up to 1 hour
- Concentration: Yes
- Short effect: Create a sphere of heavily obscuring fog.
- Quick-reference summary: Use it to block sight, escape, hide movement, or make ranged attacks difficult.
- Spell slot: 1st-level spell slot

#### Cure Wounds

- Level: 1
- Action type: Action
- Casting time: Action
- Duration: Instantaneous
- Concentration: No
- Short effect: Restore hit points to a creature you touch.
- Quick-reference summary: Heal an adjacent ally or yourself if allowed by the situation.
- Spell slot: 1st-level spell slot

## First quick-reference sheet target

Use Colossus Slayer as the first concrete quick-reference bottom-sheet example.

It should immediately show:

- Name: Colossus Slayer
- Type: Hunter feature
- Plain-language effect: After you hit an enemy that is already wounded, add 1d8 damage.
- Timing: Once per turn
- Resource: No limited use
- Reminder: The enemy must be below its hit point maximum before the hit.
- Optional detail: The bonus applies once per turn, not once per attack.
