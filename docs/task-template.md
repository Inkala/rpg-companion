# Task Template

Use this structure for new task folders.

## `PLAN.md`

```text
# T-###: Task title

Status: planning

## Goal

## Context

## Non-Goals

## Likely Files Or Artifacts

## Risks

## Validation

## Next Action
```

## `REQUIREMENTS.md`

```text
# T-### Requirements

## Problem

## Goals

## Non-Goals

## Required Behavior

## Acceptance Criteria

## Open Questions
```

## `DESIGN.md`

```text
# T-### Design

## Approach

## Files Or Artifacts

## Tradeoffs

## Risks

## Validation Plan
```

## `TASKS.md`

```text
# T-### Tasks

Status: pending

- [ ] First small reviewable item
- [ ] Second small reviewable item
```

## `NOTES.md`

```text
# T-### Notes

## Discoveries

## Open Questions

## Links
```
